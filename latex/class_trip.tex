\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_trip}{Trip}} class. An object of this type matches a trip that an user want to upload to the database. It is also used when an user wants to check the reviews for a given destination. The functionalities of this class enable writing and reading the data from the personal database files of the users.
